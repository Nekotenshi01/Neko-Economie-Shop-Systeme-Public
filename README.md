# Neko Economy - Shop System (Forge 1.20.1)

Neko Economy is a lightweight economy and marketplace mod for **Minecraft Forge 1.20.1**.  
It provides a fully configurable **Auction House (AH)** and **Shop system** with GUI, taxes, listings, and player-driven trading.

---

## Features

-  **Auction House (AH)**
  - Player listings
  - Fees & taxes
  - Duration & expiration system

-  **Shop System**
  - Categories
  - Buy/Sell prices
  - Unlimited stock

-  **Sell GUI**
  - Quickly sell items in bulk
  - Only items with a defined shop price can be sold

-  **Currency System**
  - Player balance
  - Admin controls
  - Leaderboards support

-  **Account GUI**
  - Manage listings
  - View balance

-  **Fully Configurable**
  - Messages
  - Tooltips
  - Fees & limits
  - Formatting

-  **FTB Quests Integration (Optional)**
  - Lock/unlock features with quests
  - Trigger quests via shop/AH actions

---

##  Installation

1. Install **Minecraft Forge 1.20.1**
2. Download the mod
3. Place the `.jar` file in your `/mods` folder
4. Launch the game

---

##  Commands

###  Money
- `/money` → Show your balance  
- `/money add <player> <amount>` → Add money  
- `/money remove <player> <amount>` → Remove money  

###  Auction House
- `/ah` → Open AH  
- `/ah sell <price> <days>` → Create listing  

###  Shop
- `/shop` → Open shop  

###  Selling
- `/sellgui` → Open selling interface  

###  Admin
- `/nekoecon shop <true|false> [global|player]` → Enable/disable shop  
- `/nekoecon ah <true|false> [global|player]` → Enable/disable AH

##  FTB Quests Integration
